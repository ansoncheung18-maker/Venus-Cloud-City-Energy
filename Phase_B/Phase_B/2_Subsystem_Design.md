# 金星專案 Phase B：子系統設計（Subsystem Design）

**版本：1.0**  
**更新日期：2026-05-30**

---

## 1. 太陽能子系統

### 1.1 設計目標
- 提供 500 MW 平均功率
- 抵抗硫酸腐蝕
- 適應金星高空環境（20-50°C）

### 1.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 電池類型 | GaAs 薄膜 | 單接面 |
| 安裝方式 | 水平鋪設於漂浮平台 | |
| 保護層 | PTFE 封裝 | 抗硫酸 |
| 防塵 | 無（高空無塵） | |

### 1.3 質量與功率

| 參數 | 數值 |
|:---|:---|
| 總面積 | 1 km² |
| 面密度 | 0.3 kg/m² |
| 總質量 | 300 噸 |
| 效率 | 35% |
| 總發電量 | 500 MW |

## 6. 子系統關係

### 6.1 能量流向

太陽能子系統 (500 MW)
↓
┌───┴───┐
↓       ↓
儲能系統  直接傳輸
(50 GWh) (日間)
↓       ↓
└───┬───┘
↓
傳輸系統 (450 MW)
↓
金星軌道衛星


### 6.2 子系統依賴關係

| 子系統 | 依賴 | 被依賴 |
|:---|:---|:---|
| 太陽能 | 漂浮結構 | 儲能、傳輸 |
| 漂浮結構 | 無 | 所有子系統 |
| 儲能 | 太陽能 | 傳輸 |
| 傳輸 | 儲能 | 金星軌道衛星 |
| 控制 | 太陽能 | 所有子系統 |

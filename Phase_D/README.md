## 三、關鍵發現：金星長夜問題

### 3.1 問題描述

金星夜間長達 **2 個月（約 1,440 小時）**，太陽能電廠喺呢段時間完全無法發電。所需儲能容量極大，任何現實電池都無法滿足。

### 3.2 優化方案對比

| 方案 | 儲能需求 | 可行性 |
|:---|:---|:---|
| 全電池儲能 | 極大（數千 GWh） | ❌ 不可行 |
| 日間直接傳輸 + 軌道中繼 | 大幅減少 | ✅ 可行 |
| 混合能源（太陽能 + 核能） | 0 | ✅ 可行 |

### 3.3 最終解決方案

**金星太陽能電廠定位為「日間專用」電廠，夜間由軌道中繼衛星或核能補足。**

| 時段 | 能源來源 | 說明 |
|:---|:---|:---|
| 日間（2 個月） | 太陽能電廠 | 直接傳輸至軌道衛星 |
| 夜間（2 個月） | 軌道衛星儲能 | 日間儲存嘅能量 |
